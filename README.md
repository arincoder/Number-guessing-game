# Number-guessing-game
A Python number guessing game that demonstrates loops, conditionals, and game logic.
## Features
- Difficulty levels (Easy, Medium, Hard)
- Limited attempts based on difficulty
- Replay option
- Hint system (higher/lower)
- Tracks number of tries

## What I learned
- Loops (for and while)
- Conditional statements
- User input handling
- Random number generation
- Game logic design
